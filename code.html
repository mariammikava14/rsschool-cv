Code Example from my university Quiz

Write a function that takes only one parameter name, and returns the greeting text "Hello" and the name we pass. For example, if we pass the value "Davit" as a parameter, it should return: "Hello Davit". Also, this parameter should have a default value of "You" (if you don't pass the parameter, it should return "Hello You"). Call the function with and without passing a parameter and print both cases.
Create a function that takes an array as a parameter, prints a list of distinct elements, calculates and returns their average. For example if the input array is: [1,2,3,3,3,3,4,5] The function should print [1, 2, 3, 4, 5] and return 3.
